# AIDA64报告智能解析工具

一个用于批量解析AIDA64系统检测报告的智能工具，支持自定义提取、可视化操作和多格式输出。

## 📋 功能特点

### 核心功能
- ✅ **批量处理**：支持多个AIDA64报告文件同时解析
- ✅ **自定义提取**：可视化选择需要提取的数据项目
- ✅ **模板系统**：预置多种解析模板，支持自定义模板
- ✅ **多格式输出**：导出为Excel、CSV、JSON格式
- ✅ **1:1还原**：完全按照原始报告结构输出

### 高级功能
- 🎯 **智能解析**：自动识别报告结构和编码
- 📊 **数据预览**：解析前可预览提取结果
- 🔧 **正则支持**：支持自定义正则表达式提取
- 📁 **文件夹监控**：自动监控文件夹中的新报告

## 🚀 快速开始

### 安装方法

#### 方法一：使用可执行文件（推荐）
1. 下载 `AIDA64_Report_Parser.exe`
2. 直接运行即可，无需安装Python环境

```bash
